button-app-titanium
===================

Object button used for Apps in Titanium Appcelerator.

Example:

	###
	Button: require('/ui/elements/Button');

	###
	var button1 = new Button({
	    backgroundColor: '#000',
		borderRadius: '10dp', 
		width: '200dp'
		height: '40dp'
	    top: 0
		left: '100dp',
		id: 1
   	}
